# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in the SmartCam Platform firmware, please report it privately by opening a security advisory on GitHub.

Do not disclose the vulnerability publicly until it has been addressed.

## Scope

- SmartCam OS firmware running on ESP32-S3
- SmartCam Dashboard web interface
- REST API and WebSocket endpoints
- OTA update mechanism
- Configuration storage and credentials

## Out of Scope

- Physical access to the hardware device
- Network-level attacks (Wi-Fi encryption is handled by the network infrastructure)
